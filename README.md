# Nexora Web Package

This package contains **HexLoom**, **Finder**, and **FLauncher** — a complete ecosystem for creating, managing, and running custom games; IN A GAME!

---

## Components

### 1. HexLoom
- GUI tool for making modded versions of the game.

### 2. Finder
- The game itself; every version is treated as a “mod,” including the base game. This is what makes the game fully modular.

### 3. FLauncher
- Launcher for selecting and running different versions of Finder.

### Error Code Locator
- This helps you sort out the error codes you could potentially get when using any of the applications listed above. https://clenner.github.io/nexora/errorcoder
