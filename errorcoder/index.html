Finder Error Codes

1x000
Cause: No versions found in Launcher/versions
1x001
Cause: .attr file missing from selected version
1x002
Cause: Non-folder file found in Launcher/versions
1x003
Cause: Invalid .attr JSON (malformed or unparseable)
1x004
Cause: Missing src folder in selected version
1x005
Cause: Missing required folder inside src (blocks/items/recipes/tags/dimensions)
1x006
Cause: File extension mismatch inside a data folder (e.g. dirt.item inside blocks)
1x007
Cause: Asset failed to load (pygame image load error or corrupt file)
1x008
Cause: Invalid file found in assets folder (unsupported extension)
1x009
Cause: Sound failed to load (pygame mixer error or corrupt .snd file)
1x00A
Cause: Invalid file found in sounds folder (unsupported extension)
